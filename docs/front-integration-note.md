# Intégration front réseau terrain

Le front PharmaBiz utilise désormais `src/main-v3.jsx` comme point d’entrée.

Le menu **Réseau terrain** donne accès à :

- la création et la consultation des animateurs indépendants ;
- la création et l’affectation de missions ;
- le suivi des statuts de mission ;
- la saisie du compte rendu, des ventes et du chiffre d’affaires ;
- la validation des missions réalisées.

Les tables Supabase requises sont `field_animators` et `field_missions`.
